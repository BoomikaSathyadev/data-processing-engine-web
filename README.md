# Data Processing Engine (Web)

Interactive DSA demo: **BST** with SVG tree visualization, **DFS** & **BFS** traversals, **stack undo**, **Selection Sort** & **Quick Sort**.

**Live demo:** https://boomikasathyadev.github.io/data-processing-engine-web/

## Tech stack

| Layer | Tech |
|-------|------|
| UI | HTML, CSS, JavaScript |
| Core logic | Java (`java/Engine.java`) |

## Run locally

Open `index.html` in a browser.

## Java (core logic)

```bash
cd java
javac *.java
java Engine
```

## GitHub Pages

Repo Settings → Pages → Deploy from branch `main` → folder `/ (root)`.
